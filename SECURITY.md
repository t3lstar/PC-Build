# Security Policy

This repository contains documentation for a custom PC build. It does not contain production application code or hosted service credentials.

## Reporting Issues

Use GitHub Issues to report:

- Incorrect safety guidance.
- Potentially unsafe hardware installation instructions.
- Incorrect power, thermal, BIOS, or driver recommendations.
- Broken public-site build or deployment behavior.

Do not include private account details, purchase information, serial numbers, license keys, or other sensitive information in public issues.

## Scope

Security review for this repository is limited to documentation safety, public GitHub Pages publication, and repository workflow configuration.
