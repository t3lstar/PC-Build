## Acceptable RAM Alternatives

Use a matched 2 x 16GB DDR5 kit. Do not mix separate memory kits.

Required RAM specification:

- DDR5 desktop UDIMM.
- 32GB total capacity.
- 2 x 16GB preferred.
- DDR5-6000 target speed.
- CL30 target primary latency.
- AMD EXPO profile preferred.
- Low-profile heat spreader preferred for easier cooler and cable clearance.

Acceptable examples:

| Alternative | Manufacturer reference | Amazon UK link | Notes |
| --- | --- | --- | --- |
| Corsair Vengeance 32GB `CMK32GX5M2B6000Z30` | [Corsair Vengeance DDR5-6000 CL30 AMD EXPO](https://www.corsair.com/us/en/p/memory/cmk32gx5m2b6000z30/vengeance-32gb-2x16gb-ddr5-dram-6000mt-s-cl30-amd-expo-memory-black-cmk32gx5m2b6000z30) | [Amazon UK](https://www.amazon.co.uk/CORSAIR-VENGEANCE-6000MHz-Compatible-Computer/dp/B0CBRJ63RT) | Meets the DDR5-6000 CL30 EXPO target. Check motherboard QVL before purchase. |
| G.SKILL Flare X5 `F5-6000J3038F16GX2-FX5` | [G.SKILL Flare X5](https://www.gskill.com/product/165/396/1673491242/F5-6000J3038F16GX2-FX5) | [Amazon UK](https://www.amazon.co.uk/G-SKILL-2x16GB-CL30-38-38-96-Desktop-Computer-matte-black/dp/B0C1TKSDKR) | Preferred kit. Low-profile 33mm module height. |

Avoid:

- DDR4 memory. It is physically incompatible with this motherboard.
- Four-DIMM kits unless there is a specific reason and the memory support list confirms the target speed.
- Non-EXPO kits when an EXPO kit is available at a similar price.
- Mixing separate 16GB modules or separate 2 x 16GB kits.

## Acceptable SSD Alternatives

Use an M.2 2280 NVMe SSD. PCIe 4.0 x4 is sufficient for this build; PCIe 5.0 drives are acceptable but not required.

Required SSD specification:

- M.2 2280 form factor.
- NVMe protocol.
- 2TB target capacity.
- PCIe 4.0 x4 or better.
- TLC NAND preferred.
- Suitable thermal behavior under a motherboard heatsink.

Acceptable examples:

| Alternative | Manufacturer reference | Amazon UK link | Notes |
| --- | --- | --- | --- |
| Samsung 990 PRO 2TB `MZ-V9P2T0` | [Samsung 990 PRO 2TB](https://www.samsung.com/us/memory-storage/nvme-ssd/990-pro-pcie-4-0-nvme-ssd-1tb-sku-mz-v9p2t0b-am/) | [Amazon UK search](https://www.amazon.co.uk/s?k=Samsung+990+PRO+2TB+MZ-V9P2T0B) | Higher-performance PCIe 4.0 alternative. Use a motherboard heatsink unless buying a heatsink version intentionally. |
| WD_BLACK SN850X 2TB | [WD_BLACK SN850X](https://www.sandisk.com/products/ssd/internal-ssd/wd-black-sn850x-nvme-ssd) | [Amazon UK](https://www.amazon.co.uk/WD_BLACK-SN850X-NVME-SSD-NON-HEATSINK/dp/B0B3RQDGL7) | Strong PCIe 4.0 alternative. Prefer the non-heatsink model when using the motherboard heatsink. |

Avoid:

- SATA M.2 drives.
- 2230, 2242, or 2260 drives unless deliberately using an adapter or a secondary slot that supports that length.
- Unknown-brand SSDs without clear endurance, warranty, and controller information.
- Tall third-party heatsinks that conflict with motherboard M.2 heatsinks.

## Pre-Purchase Verification Checklist

- Confirm each retailer listing matches the exact model number in this page.
- Confirm the motherboard revision and BIOS support before assembly.
- Check the Gigabyte memory support list for the chosen RAM kit.
- Confirm the case colour and variant: `O11DMIV2FX` for black Flow.
- Confirm the GPU listing is the 16GB model, not the 8GB model.
- Confirm the SSD listing is the 2TB model.
- Confirm the PSU listing is the ASUS TUF Gaming 1000W Gold, not a lower-wattage variant.
- Keep invoices and serial numbers outside the public repository.
