# Gaming Optimisation

Status: Initial Milestone 2 content. Last verified: 2026-07-13 10:53 BST.

## Introduction

This chapter applies practical Windows and Radeon settings for a stable gaming PC after drivers are installed.

## Purpose

Improve gaming readiness without unsafe overclocking or unnecessary background software.

## Estimated Time

30-60 minutes.

## Difficulty

Beginner.

## Required Tools

- Windows administrator account.
- AMD Radeon Software.
- Game launcher accounts.
- Monitor manual if adjusting refresh rate.

## Warnings

- Do not overclock the CPU or GPU as part of baseline setup.
- Do not disable security features for minor benchmark gains.
- Do not install multiple RGB, motherboard, or tuning utilities unless needed.
- Change one performance setting at a time.

## Step-by-Step Instructions

1. Confirm Windows Update and drivers are complete.
2. Set the monitor to its native resolution and maximum supported refresh rate.
3. Enable Windows Game Mode.
4. Confirm Hardware-Accelerated GPU Scheduling only if it is stable with the installed driver.
5. Open AMD Radeon Software and confirm the GPU is detected correctly.
6. Leave GPU tuning on default for the baseline.
7. Set game install libraries to the NVMe SSD.
8. Disable unnecessary startup applications.
9. Configure Windows power settings for balanced or high-performance use as appropriate.
10. Create a restore point before installing large game launchers or utilities.

## Verification Checklist

- [ ] Monitor uses the intended resolution.
- [ ] Monitor uses the intended refresh rate.
- [ ] AMD Radeon Software detects the GPU.
- [ ] GPU tuning remains at default baseline settings.
- [ ] Game Mode is enabled.
- [ ] Startup apps are reviewed.
- [ ] A restore point exists.

## Common Mistakes

- Benchmarking before setting monitor refresh rate.
- Applying GPU undervolts or overclocks before baseline testing.
- Installing several motherboard utilities that duplicate functions.
- Disabling Windows security features unnecessarily.
- Testing with background downloads still running.

## Expected Result

The system is configured for gaming with a clean baseline suitable for benchmark comparison.
