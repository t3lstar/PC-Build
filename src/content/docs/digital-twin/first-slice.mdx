If JavaScript is unavailable, this page must still explain the state of the digital twin data:

- Exact connector coordinates remain pending or partially verified.
- Cable routes are represented as source data before detailed path interaction is added.
- Airflow has text fallback content before animation is added.
- The case view includes a static zone list.
- The connector map includes a static connector list.
